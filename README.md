## 学习建议
- `确定学习目标，制定学习计划` 如学习JVM，每周一个小目标
- `抓住学习主线任务，不要在其它支线任务浪费时间` 如学习Kafka，如果你是后端开发，就不要纠结怎么把环境搭建的完美，先跑起来再说
- `定期复习和总结` 定期复盘，优化学习路线和方法，定期复习，加深记忆
